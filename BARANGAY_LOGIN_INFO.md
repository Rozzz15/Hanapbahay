## How to Login

1. Open the app and go to the **Login** screen
2. Enter one of the emails above
3. Enter the corresponding password
4. Click **"Sign In"**
5. You will be automatically redirected to the **Barangay Dashboard**

## What You'll See

After logging in, you'll have access to:

### Dashboard Overview
- **Total Residents**: Count of registered users in your barangay
- **Total Properties**: Number of available rental listings
- **Active Listings**: Published property listings
- **Active Bookings**: On-going rental agreements

### Navigation Menu (Bottom Bar)
- **Dashboard**: Main overview with statistics
- **Residents**: View and manage residents
- **Properties**: Browse available rentals
- **Reports**: Generate analytics
- **Settings**: Manage your preferences

### Design Features
- Modern, professional interface matching the Owner Dashboard
- Clean card-based layouts
- Responsive grid system
- Green color scheme (#10B981)
- Intuitive bottom navigation
